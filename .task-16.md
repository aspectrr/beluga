# Task 16: Agent Loop, LLM Client, Context Builder, Compactor

## Status: Complete

All four files created and verified:

### Files Created

1. **internal/core/agent/llm.go** — OpenAI-compatible LLM client
   - `LLMClient` with `Chat()` (non-streaming) and `ChatStream()` (SSE streaming)
   - Types: `ChatMessage`, `ToolCall`, `FunctionCall`, `ToolDef`, `FunctionDef`, `ChatRequest`, `ChatResponse`, `ChatChoice`, `ChatUsage`, `StreamChunk`
   - `NewLLMClient(cfg)` from `config.LLMConfig`
   - Bearer token auth, POST to `/chat/completions`

2. **internal/core/agent/context.go** — Event-to-message converter and system prompt assembly
   - `ContextBuilder` with `Build(events) → []ChatMessage`
   - Handles compacted events (uses summary as system prompt context, skips compacted events in output)
   - `eventToMessages()` handles all event types: user_message, agent_message, tool_call, tool_result, interrupt, error
   - `truncate()` drops oldest messages (keeping system prompt) when over token budget
   - `EstimateTokens()` using chars/4 heuristic
   - Constants: `MaxContextTokens=128000`, `charsPerToken=4`

3. **internal/core/agent/compactor.go** — Context compaction when approaching token limit
   - `Compactor` with `NeedsCompaction()` and `Compact()`
   - Compaction threshold: 85% of max tokens
   - Target: keep 40% of context, summarize 60%
   - Clean boundary detection (never cuts mid tool-call/result pair)
   - Calls LLM to summarize, appends `compacted` event
   - Constants: `CompactionThreshold=0.85`, `TargetCompactionRatio=0.40`

4. **internal/core/agent/loop.go** — Main agent orchestrator
   - `Orchestrator` with `StartRun()`, `Interrupt()`, `Resume()`, `Terminate()`, `HandleNewSession()`
   - `ToolExecutor` interface decoupled from tools package
   - `runLoop()`: max 30 iterations, each builds context → checks compaction → calls LLM → handles tool calls or agent message
   - `SetTools()` wires tool definitions for LLM function calling
   - `mustJSON()` and `usageTokens()` helpers (used by compactor too)
   - `NewWithPrompt()` for custom system prompt (will be used with SYSTEM.md content)
